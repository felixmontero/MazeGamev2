# MazeGame v2

## 🇪🇸 Español
¡Hola! Bienvenido a **MazeGame v2**. Este proyecto es una versión en JavaScript (usando **React** y **HTML5 Canvas**) de mi anterior proyecto del Grado Superior, el cual desarrollé originalmente en Java. He querido coger ese mismo concepto y llevarlo a la web, dándole un lavado de cara con un diseño mucho más moderno y mecánicas más pulidas.

### ¿Qué incluye esta versión?
- **Exploración de Salas:** Te puedes mover por las distintas salas del laberinto usando los controles en pantalla o el teclado (WASD / Flechas).
- **Sistema de Objetos:** Por el mapa hay repartidas monedas 🪙 que puedes ir recogiendo.
- **Gestión de Llaves y Puertas:** Las monedas te sirven para "comprar" llaves 🔑 y así poder desbloquear puertas cerradas 🔒.
- **Animaciones Pixel Art:** He integrado un sistema de sprites para el personaje principal (con animaciones al caminar en las 4 direcciones), dándole un toque clásico.
- **Apertura de Puertas Automática:** Ahora el gameplay es mucho más fluido. Si tienes una llave, al chocar contra una puerta cerrada, esta se abre de forma automática consumiendo la llave.
- **Diseño UI Moderno:** He implementado un tema oscuro ("Dark Mode") basándome en el estilo "Glassmorphism", con notificaciones flotantes que no rompen el diseño ni mueven los elementos de la pantalla.

### Cómo probarlo en local
1. Clona este repositorio.
2. Abre la terminal en la carpeta del proyecto.
3. Ejecuta `npm install` para instalar las dependencias necesarias.
4. Ejecuta `npm run dev` para lanzar el servidor de Vite.
5. Abre en tu navegador la URL que te indique (suele ser `http://localhost:5173/`).
6. Dale a "Jugar" y ¡encuentra la salida!

## 🇬🇧 English
Hi! Welcome to **MazeGame v2**. This project is a JavaScript version (built with **React** and **HTML5 Canvas**) of my previous Higher Vocational Training (Grado Superior) project, which I originally developed in Java. I wanted to take that core concept and bring it to the web with a much more modern design and polished mechanics.

### What's new in this version?
- **Room Exploration:** You can navigate through the labyrinth rooms using the on-screen controls or your keyboard (WASD / Arrow Keys).
- **Item System:** There are coins 🪙 scattered across the map for you to collect.
- **Keys and Doors Management:** You use the coins to "buy" keys 🔑 and unlock closed doors 🔒.
- **Pixel Art Animations:** I've integrated a sprite system for the main character (with 4-directional walking animations), giving it a classic retro feel.
- **Automatic Door Unlocking:** Gameplay is now much more fluid. If you have a key, simply walking into a locked door will automatically open it and consume the key.
- **Modern UI Design:** I've implemented a premium Dark Mode based on the "Glassmorphism" style, featuring smooth toast notifications that don't cause any layout shifts.

### How to run it locally
1. Clone this repository.
2. Open a terminal in the project folder.
3. Run `npm install` to install the dependencies.
4. Run `npm run dev` to start the Vite server.
5. Open the provided URL in your browser (usually `http://localhost:5173/`).
6. Click "Jugar" (Play) and find the exit!
